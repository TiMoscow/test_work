# Тестовая площадка

Проект разработан как тестовая площадка по тестированию кода. 
Проект приспособлен для частного использования и в настоящий момент не приспособлен для онлайн формата в открытом доступе.
Конвертации изображения.


### Преимущества 
* Создание тестового кода
* Конвертация изображения в формат *.webp *
* Создание новой страницы с интегрированным API

### Параметры конвертора изображения
| Класс / методы     |   Тип  | Описание
| ------------------- | ------ | ------------------------------------------------------
| Imgwebp             | Класс  | Класс конвертора
| get_browser_name    | Метод  | Проверка на поддержку браузерами формата *.webp
| function_img_webp   | Метод  | Метод преобразования изображений

### Параметры работы с файлами
/user_library/safe_file.php

| Класс / методы     |   Тип  | Описание
| ------------------- | ------ | ------------------------------------------------------
| operations_file     | Класс  | Класс по работе с файлами.
| safe_file_search    | Метод  | Проверка файлов/директорй, создание файлов/директорий.
| delite_file         | Метод  | Проверка файлов, удаление файлов.
| checking_the_file   | Метод  | Прием свойств, проверка, сообщения.
